# Synth Hits

A collection of drum sounds sampled from Yamaha CS30L, Roland Jupiter 4, Korg Mono/Poly, Oberheim Xpander & Roland SH-2.

 - **Files**: 450+
-  **Formats:** AIF & WAV
-   **Approx Size:** 250mb
- [Web Audio Demo](https://www.modularsamples.com/Demos/demos/synthhits.html)
